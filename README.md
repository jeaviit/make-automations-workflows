# Make.com Automation Workflows

Make.com and n8n workflow templates by Jeaviit Bhaat — shared as free resources.

## Workflows

| Folder | Automation | Description |
|--------|------------|-------------|
| newsletter-to-mp3 | Newsletter to MP3 | Summarizes daily newsletters and converts them to audio (MP3) via Google Drive |
| linkedin-dm-automation | LinkedIn AutoDM | Automated LinkedIn DM workflows |
| info-post-call | Discovery Call Info Post | Automation for discovery call info posts |
| product-scraping | Product Scraping | Scrape product data automation |
| blog-generator | Blog Generator | Complex blog generation automation |
| lead-comment-automation | Lead Comment Automation | LinkedIn lead generation via auto-commenting |
| lead-magnet-automation | Lead Magnet Automation | Lead magnet delivery automation |
| youtube-content-generator | YouTube Content Generator | YouTube content generation workflow |
| data-extractor | Data Extractor | Extract data from PDFs automation |

## How to Import (Make.com)

1. Open [Make.com](https://make.com)
2. Go to **Scenarios** → **Create a new scenario**
3. Click the **three-dot menu** → **Import Blueprint**
4. Upload the `.json` file from the relevant folder

## How to Import (n8n)

1. Open your n8n instance
2. Go to **Workflows** → **Add workflow**
3. Click the **menu** → **Import from file**
4. Upload the `.json` file from the relevant folder
